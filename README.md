# Find The Word - Python Game

A simple and fun "Treasure Hunt" style game written in **Python**.

## 🚀 How it works
The program will ask you if you want clues. For every "no", a more specific clue will be revealed. 
But be careful: you have limited chances to guess the secret word!

## 🛠️ Features
- **Interactive logic**: Uses nested `if/else` statements to create a game flow.
- **Input handling**: Uses `.lower()` to make sure the game understands you even if you use capital letters.
- **Clues system**: Three different levels of hints, from cryptic to very specific.
